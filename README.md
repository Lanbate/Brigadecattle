### 如何使用？
:heavy_check_mark: 安装vuecli脚手架
`npm install -g @vue/cli`

:heavy_check_mark: 安装依赖
`npm install`

:heavy_check_mark: 项目启动
`npm run serve`


:red_circle: 打包
`npm run build`
